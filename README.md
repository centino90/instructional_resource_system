# Instructional Resource System
a system for managing and distributing instructional resources 

## Setup

### Step 1

```bash
composer install
```

### Step 2

```bash
composer update
```

### Step 3

```bash
php artisan fresh:all
```

### Step 4

```bash
php artisan serve
```

### Step 5

go to the url http://127.0.0.1:8000/
